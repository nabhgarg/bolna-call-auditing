## Vision

### Positioning insight (from landscape scan)
Voice-agent eval is crowded with automation-first platforms (Coval — YC-backed,
simulation; Hamming — explicitly NO human review, claims "95-96% agreement with
human evaluators"; Cekura, Roark). Every automated judge needs a defensible human
ground truth to calibrate against, and LLM judges are weakest exactly where our
data lives: Hindi/Hinglish, code-switched, Indian telephony audio. RealLoop is
NOT "human QA with a dashboard" (Coval's side feature) — it is **the calibrated
ground-truth layer for voice AI, starting where auto-evals fail worst.** Golden
transcripts compound into a benchmark/data asset.

### The 4-beat demo story (user-locked)
1. Reviewer value: tool enforces error reduction / less subjectivity + efficiency
   (structured rubric, timestamped issue marking, measured 3-4 min/call).
2. Pre-scan layer: AI first pass makes marking issues simple (L1 evidence + roadmap).
3. Rubric / evaluation framework (live in tool: vibe rubric + 3-issue taxonomy).
4. Client side: trust + output analytics + golden transcript + issues + way forward
   (= /dashboard + downloadable report).

### Demo delivery (user-decided)

- Everything else is shown in the **demo video** (Loom over live product).
